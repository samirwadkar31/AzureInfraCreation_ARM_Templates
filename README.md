# ARM-Templates

We can create AZURE Infrastructure using ARM templates. (IAC)
Ways to create Azure Infra using ARM teamplates,
1) Azure DevOps CICD Pipelines
2) Azure Custom Templates Deployment
3) Azure CLI 
4) VS Code
